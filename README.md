# go-btfs-pinner

## Background

The pinner system is responsible for keeping track of which objects a user wants to keep stored locally

## Install

Via `go get`:

```sh
$ go get github.com/TRON-US/go-btfs-pinner
```

> Requires Go 1.13

## Documentation
